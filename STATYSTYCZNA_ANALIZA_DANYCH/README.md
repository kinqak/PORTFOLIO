# Analiza wielowymiarowa danych

Repozytorium zawiera projekt analityczny dotyczący analizy wielowymiarowej danych z wykorzystaniem metod redukcji wymiarowości i wizualizacji struktury danych.

## Zakres projektu

W projekcie przeprowadzono:

- przygotowanie i eksploracyjną analizę danych,
- analizę głównych składowych (PCA),
- analizę metod skalowania wielowymiarowego (MDS),
- klasyczne skalowanie metryczne,
- nieliniowe mapowanie Sammona,
- porównanie wyników skalowania wielowymiarowego dla dwóch wymiarów,
- porównanie wyników skalowania wielowymiarowego dla trzech wymiarów,
- interpretację i podsumowanie wyników.

## Zawartość repozytorium

- `SAD.Rmd` – plik źródłowy analizy w R Markdown
- `SAD.html` – raport w formacie HTML
- `data.csv` – dane wejściowe wykorzystane w projekcie
