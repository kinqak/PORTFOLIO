# Analiza miar ryzyka rynkowego

Repozytorium zawiera projekt analityczny dotyczący estymacji i porównania wybranych miar ryzyka rynkowego na podstawie danych giełdowych.

## Zakres projektu

W projekcie przeprowadzono:
- analizę szeregów czasowych danych finansowych,
- estymację miar ryzyka **Value at Risk (VaR)** oraz **Expected Shortfall (ES)**,
- porównanie różnych metod estymacji,
- testy wsteczne modeli ryzyka, w tym test Kupca.

## Zawartość repozytorium

- `IMRR-PROJEKT.Rmd` – plik źródłowy raportu w R Markdown
- `IMRR-PROJEKT.html` – raport HTML
- `apple_daily.csv` – dane wejściowe wykorzystane w analizie
