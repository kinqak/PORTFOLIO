# Portfolio

Repozytorium zawiera wybrane projekty z zakresu analizy danych, ekonometrii, statystyki oraz uczenia maszynowego.

## Projekty

### 1. COMPARATIVE_ANALYSIS_RNN_STOCK_INDEX_PRICE_PREDICTION
Projekt dotyczący prognozowania indeksów giełdowych z wykorzystaniem sieci RNN, LSTM i GRU.

### 2. EKONOMETRIA_DANYCH_PANELOWYCH
Projekt zespołowy z zakresu ekonometrii danych panelowych dotyczący analizy zależności między stopą bezrobocia a wybranymi czynnikami społeczno-ekonomicznymi w krajach Unii Europejskiej.

### 3. EKONOMETRYCZNY_MODEL_STOPY_BEZROBOCIA
Projekt ekonometryczny dotyczący modelowania stopy bezrobocia z wykorzystaniem klasycznych metod regresji i diagnostyki modelu.

### 4. ILOŚCIOWE_MIARY_RYZYKA_RYNKOWEGO
Projekt analityczny dotyczący estymacji i porównania wybranych miar ryzyka rynkowego na podstawie danych finansowych.

### 5. STATYSTYCZNA_ANALIZA_DANYCH
Projekt z zakresu analizy wielowymiarowej danych, obejmujący m.in. PCA oraz skalowanie wielowymiarowe.

### 6. UCZENIE_MASZYNOWE
Projekt zespołowy dotyczący predykcji decyzji rekrutacyjnych z wykorzystaniem wybranych metod uczenia maszynowego.

## Wykorzystywane narzędzia

W projektach wykorzystano m.in.:
- R
- R Markdown
- Python
- Gretl
- Jupyter Notebook
- Excel

## Uwagi

Część projektów była realizowana indywidualnie, a część zespołowo. W opisach poszczególnych folderów zostało wskazane, jeśli projekt był tworzony wspólnie z inną osobą.
