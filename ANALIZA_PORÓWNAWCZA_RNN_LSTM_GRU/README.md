# Prognozowanie indeksów giełdowych z wykorzystaniem rekurencyjnych sieci neuronowych

Repozytorium zawiera projekt dotyczący prognozowania indeksów giełdowych z wykorzystaniem rekurencyjnych sieci neuronowych, w tym modeli RNN, LSTM i GRU.

## Zakres projektu

W projekcie przeprowadzono:

- przygotowanie i wstępną analizę danych,
- budowę modeli RNN, LSTM i GRU,
- trenowanie i testowanie modeli predykcyjnych,
- porównanie skuteczności poszczególnych architektur sieci neuronowych,
- ocenę jakości prognoz na podstawie wybranych miar,
- analizę wyników z wykorzystaniem wykresów i wizualizacji.

## Zawartość repozytorium

- pliki z danymi wejściowymi,
- kod źródłowy wykorzystany do budowy i trenowania modeli,
- wyniki predykcji,
- raport / praca końcowa w formacie PDF.

## Wykorzystane narzędzia

- Python
- Jupyter Notebook
- biblioteki do analizy danych i uczenia maszynowego
